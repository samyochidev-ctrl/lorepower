Quick Lore

โลก TBOTE ถูกขับเคลื่อนด้วย “แกนพลัง” ที่ทำหน้าที่เหมือนหัวใจของผู้ใช้พลัง ยิ่งแกนแข็งแรง ยิ่งควบคุมพลังได้ลึกและเสถียร

Core Mana Awakening Bloodline

RP Ready

เหมาะสำหรับเอาไปสร้าง Discord Roleplay / Web RPG / ระบบบอท โดยมีหมวด “Combat” และ “Status” ให้ใช้จริง

แกนพลัง (Core)

แกนพลังคือ “ศูนย์กลางของพลัง” ทำหน้าที่สร้าง เก็บ และแปลงพลังภายใน

1) แกนพื้นฐาน (Natural Core)

แกนพลังที่เกิดขึ้นตามธรรมชาติเมื่อมนุษย์/เผ่าเริ่ม “ตื่นพลัง” เป็นแกนที่เสถียรที่สุด แต่เติบโตช้า

2) แกนกลายพันธุ์ (Mutated Core)

เกิดจากเหตุการณ์พิเศษ เช่น พลังต้องห้าม, การทดลอง, การปนเปื้อนพลัง เติบโตเร็ว แต่เสี่ยง “แตกแกน” หรือสูญเสียสติ

3) แกนเลือด (Bloodline Core)

แกนที่ถูกส่งต่อทางสายเลือดของตระกูลพิเศษ มักมี “ความสามารถเฉพาะ” เช่น ไฟดำ, เวลา, มิติ

4) แกนเทียม (Artificial Core)

แกนที่ถูกสร้างขึ้นด้วยเทคโนโลยี/พิธีกรรม ใช้แทนแกนจริงได้ แต่ต้องใช้ “ตัวช่วยเสริม” ตลอดเวลา

กฎสำคัญของแกนพลัง

  • แกนพลังมี “ความจุ” และ “ความเสถียร”
  • การฝืนใช้เกินขีดจำกัด → แกนร้าว → แตกแกน → ตาย/พิการ
  • แกนพลังสามารถ “วิวัฒน์” ได้เมื่อผ่านจุดตื่นรู้

ระดับพลัง (Power Ranks)

ระบบยศที่เอาไปใช้ใน Roleplay ได้จริง (ยืดหยุ่นปรับเพิ่มได้)

เพิ่งเริ่มมีแกนพลัง ใช้พลังได้แค่พื้นฐาน (เสริมแรงกาย/ปล่อยพลังเล็กน้อย)

เริ่มสร้างทักษะได้ ใช้พลังต่อเนื่องได้มากขึ้น แต่ยังไม่เสถียร

สามารถต่อสู้จริงได้ ใช้สกิลได้หลายแบบ และเริ่มมี “สายพลัง” ชัดเจน

เริ่มมีเทคนิคขั้นสูง เช่น การกดแกน, การเสริมแกน, การสร้างสนามพลัง

สามารถทำลายพื้นที่ขนาดใหญ่ มีสกิลเอกลักษณ์ระดับสูง

ระดับที่องค์กรต่างๆต้องการตัว สามารถคุมสนามรบได้คนเดียว

ระดับตำนาน พลังเหนือกฎของโลก มีอิทธิพลต่อยุคสมัย

เผ่า (Races)

โลก TBOTE สามารถมีหลายเผ่า โดยแต่ละเผ่าจะมีแกนพลังและข้อจำกัดต่างกัน

มนุษย์ (Human)

สมดุลที่สุด พัฒนาได้ทุกสายพลัง แต่ต้องใช้เวลาในการฝึกและตื่นรู้

อสูร/ครึ่งอสูร (Demon / Half)

มีแกนพลังดิบสูงมาก แข็งแรง แต่เสี่ยง “พลังกลืนจิต” หากควบคุมไม่ได้

เผ่าจิตวิญญาณ (Spiritborn)

เชื่อมต่อพลังมิติ/ธรรมชาติได้ง่าย แต่พลังอาจลดลงเมื่ออยู่ในพื้นที่ปิดตาย

เผ่าเทียม (Artificial Race)

ถูกสร้างด้วยการทดลอง มีแกนเทียม พลังพุ่งเร็ว แต่มี “อายุแกน” จำกัด

องค์กรหลัก (Organizations)

โครงสร้างโลกที่เหมาะมากสำหรับทำ Roleplay และ Campaign

1) สมาพันธ์นักล่า (Hunter Federation)

หน่วยงานที่จัดอันดับนักสู้ รับเควส กำจัดภัยคุกคาม มีระบบยศ, เงินรางวัล, และพื้นที่ควบคุม

2) สถาบันฝึกแกน (Core Academy)

โรงเรียน/สถาบันที่ฝึกผู้ตื่นพลัง มีการสอบ, การประเมินแกน, และการฝึกสนามจริง

3) องค์กรวิจัยแกน (Core Research Lab)

ศึกษาแกนเทียม, การวิวัฒน์แกน, และพลังต้องห้าม เป็นแหล่งกำเนิดเหตุการณ์ดาร์คหลายอย่าง

4) กลุ่มต้องห้าม (Forbidden Cult)

กลุ่มที่เชื่อว่าการ “แตกแกน” คือทางสู่พลังสูงสุด ใช้พิธีกรรมและการสังเวย

ระบบต่อสู้ (Combat System)

ระบบต่อสู้ที่คุณสามารถเอาไปใช้กับ Discord Dice / D&D ได้ทันที

ค่าสเตตัสพื้นฐาน

  • STR (พลังโจมตี)
  • AGI (ความเร็ว/หลบ)
  • VIT (ความทนทาน)
  • INT (ควบคุมพลัง)
  • CORE (ความจุแกน)

สถานะสำคัญ

  • Core Burn (แกนไหม้)
  • Core Crack (แกนร้าว)
  • Mana Poison (พิษพลัง)
  • Overdrive (โอเวอร์ไดรฟ์)

แนวคิดการต่อสู้

  • พลังไม่ใช่ทุกอย่าง → “เทคนิค” สำคัญกว่า
  • ผู้ชนะมักเป็นคนที่คุมแกนได้ดีที่สุด
  • การแตกแกนเป็นความเสี่ยงระดับชีวิต

สายพลัง (Power Paths)

สายพลังคือแนวทางการใช้แกนพลังที่แตกต่างกัน

สายเสริมร่าง (Body Reinforcement)

เสริมกล้ามเนื้อ กระดูก ประสาท เพิ่มความเร็วและพลังโจมตี

สายธาตุ (Elemental)

ไฟ น้ำ ลม ดิน น้ำแข็ง สายฟ้า ฯลฯ ใช้คุมสนามรบ

สายมิติ (Spatial)

บิดมิติ เปิดช่องว่าง วาร์ป ตัดพื้นที่ เป็นสายที่หายาก

สายจิต (Mental)

ควบคุมจิต อ่านความคิด สร้างภาพลวงตา หรือโจมตีจิตวิญญาณ

สายต้องห้าม (Forbidden)

ใช้พลังแลกกับอายุขัย สติ หรือร่างกาย มักเกิดจากพิธีกรรม
